# idea. - App de capture d'idees

## API

La documentation complete de l'API est disponible en ligne :

```
GET https://ton-domaine.com/api/docs
```

En local :
```
GET http://localhost:3007/api/docs
```

Lis cette URL avant d'appeler l'API pour avoir les parametres exacts, les types acceptes et des exemples.

## Cle API

Genere ta cle depuis `/settings` dans l'app, puis :

```bash
export IDEA_API_KEY="idea_sk_..."
```
